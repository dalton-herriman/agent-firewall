# Examples

## SDK

Run:

```bash
python examples/sdk_guarded_tool.py
```

## LangChain-style wrapper

Run:

```bash
python examples/langchain_wrapper.py
```

## OpenAI Agents-style wrapper

Run:

```bash
python examples/openai_agents_wrapper.py
```

## Bootstrap

Use:

```bash
make bootstrap
```

This copies `.env.example` to `.env` when needed and prints the included development seed counts from `seed/dev_seed.json`.
